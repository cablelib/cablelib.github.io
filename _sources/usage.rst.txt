Usage
======

Installation
-------------

The python module is hosted in this project's PyPi package register. Install it
with

.. code-block::

    pip install cablelib-isw-pybindings --index-url https://__token__:<your_personal_token>@git.isw.uni-stuttgart.de/api/v4/projects/2340/packages/pypi/simple

Your personal access token needs to have at least ``api-read`` permissions.
