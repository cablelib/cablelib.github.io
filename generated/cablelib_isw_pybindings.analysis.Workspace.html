Bases: pybind11_object

Class for the workspace of CDPRs.

__init__(self: cablelib_isw_pybindings.analysis.Workspace, cdpr: cablelib_isw_pybindings.robot.CableDrivenParallelRobot, sphere_approx_depth: int = 3) → None

Constructs a workspace object.

On construction of a workspace object a unit sphere is approximated as preparation for the line search algorithm.

Parameters:

  • cdpr (cablelib_isw_pybindings.robot.CableDrivenParallelRobot) – The CDPR for which the workspace is to be analyzed

  • sphere_approx_depth (int) – The depth to which the unit sphere is approximated, defaults to 3

Raises:

ValueError – if the sphere approximation depth is negative.
