class cablelib_isw_pybindings.robot.Cable

Bases: pybind11_object

Class which models a cable of the CDPR.

__init__(self: cablelib_isw_pybindings.robot.Cable, min_force: float, max_force: float, young_modulus: float = 110000000000.0, cross_section: float = 0.00011309733553, constant_length: float = 0.0) → None

Constructor for a cable.

If no technical parameters of the cable are supplied, it is assumed that the cable is a steel cable.

Parameters:

  • min_force (float) – Minimum force of the cable [N]

  • max_force (float) – Maximum force of the cable [N]

  • young_modulus – Young modulus of the cable [Pa]. Default is 100 GPa.

  • cross_section – Cross section of the cable [m^2]. Default is 0.00011309733553 m^2.

  • constant_length – The constant length of the cable [m]. Default is 0.
