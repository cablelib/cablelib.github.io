static load_wcrfx(filepath: str) → cablelib_isw_pybindings.robot.CableDrivenParallelRobot

Loads a legacy .wcrfx file and returns a CableDrivenParallelRobot object

.wcrfx files are used in the WireX project to save the properties of a CDPR. However, these files do not adhere to the standard xml file format and cannot be parsed non trivially.

Parameters:

filepath (str) – The path of the .wcrfx file

Raises:

ValueError – if the filepath does not point to a .wcrfx file

Returns:

Instantiated CableDrivenParallelRobot object from given .wcrfx file

Return type:

cablelib_isw.Robot.CableDrivenParallelRobot
