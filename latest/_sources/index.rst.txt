cablelib_isw_pybindings documentation
===================================================

**cablelib_isw_pybindings** is a Python module for
Cable-Driven Parallel Robots (CDPRs).
This module offers methods to model and analyze CDPRs.

The algorithms are implemented in C++ to ensure an efficient
execution and bound with `nanobind <https://github.com/wjakob/nanobind>`__
to make them callable with Python code for ease of use.

This project focuses on an extendable object-oriented structure
which follows SOLID principles to manage the ever-growing complexity
in analysis tasks.

Features of **cablelib_isw_pybindings** include:

* Modelling of CDPRs, exportable in .xml format
* Force distribution calculation with various algorithms
* Workspace calculation with custom workspace criteria either as
  hull or discrete grid
* System matrices, i.e. the CDPR's structure matrix or stiffness matrix
* Inverse and Forward Kinematics
